## 🚀 Getting Started

### Prerequisites
* Python 3.8 or later
* FastAPI 0.115.0 or later
* Uvicorn 0.30.6 or later
* SQLAlchemy 2.0.35 or later
* Asyncpg 0.29.0 or later

### Installation
```bash
pip install fastapi uvicorn sqlalchemy asyncpg
```

### Quick Start
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```
