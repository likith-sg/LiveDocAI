                        How Drift Detection Works
                    

                    

                        

                            1.
                            
Code diff analysis — compares latest commit with previous to see what changed

                        

                        

                            2.
                            
Docs comparison — checks if existing README/docs still accurately describe the new code

                        

                        

                            3.
                            
Auto PR — when drift found, generates updated docs and opens a PR automatically
